# java-lotto 
로또 미션 진행을 위한 저장소 
## 1단계-자동
### TODO List
1. 로또 구입 금액을 입력받는다.
   - 로또 1장의 가격은 1000원이다.
   - [예외] 입력이 정수가 아닌 경우
   - [예외] 입력이 0이하인 경우 
   - [예외] 구입 금액이 1000의 배수가 이닌 경우
2. 로또 구매 개수를 출력한다.
3. 구매한 로또 목록을 출력한다.
4. 지난 주 당첨 번호를 입력받는다.
   - [예외] 입력 형식 '(([0-9]+)(,)( )){5}([0-9]+)' 이 아닌 경우
   - [예외] 각각의 수가 1~45를 벗어난 경우
   - [예외] 중복 숫자가 있는 경우
5. 당첨 통계를 출력한다.
6. 수익률을 출력한다.
